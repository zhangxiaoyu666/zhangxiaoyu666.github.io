List接口的实现类：ArrayList

底层结构：可变数组

  • 底层

ArrayList 初始维护了一个Object类型的数组，数组长度为0

当添加第一个元素的时候，elementData数组初始化长度为10

添加10个元素后，再继续添加元素，elementData数组长度扩容为15

当15个元素添加结束后，再继续添加时，elementData数组扩容为22

所以得出结论，ArrayList每次扩容1.5倍

每个结点中有三个属性值（item,next,prev）分别指向当前元素、下一个、上一个元素，最终实现手拉手的链表结构
